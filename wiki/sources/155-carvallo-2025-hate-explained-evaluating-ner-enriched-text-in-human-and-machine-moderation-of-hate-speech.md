# Carvallo 等 - 2025 - Hate Explained Evaluating NER-Enriched Text in Human and Machine Moderation of Hate Speech

## Metadata

## Problem Framing
- Studies whether explicit semantic tags for intent and group mentions improve hate speech detection and moderation.
- Separates machine classification performance from human moderator utility, asking whether explanations help users make better decisions.
- Directly addresses ambiguity in hate moderation, especially cases where target group mentions and intent matter more than surface toxicity.

## Method
- Uses GPT-4o to generate intent and group tags for training partitions, guided by annotation guidelines.
- Trains RoBERTa-large NER models to identify intent and group spans, then inserts those tags into text as enrichment.
- Fine-tunes HateBERT classifiers on original or enriched text.
- Runs a user study with 100 moderators over seven visualization settings: intent tags, group tags, model probabilities, randomized counterparts, and controls.

## Data and Evaluation Setup
- Datasets include IHC, SBIC, and DH.
- Intent tags cover categories such as derogation, threats, hate crimes, comparisons, and animosity.
- Group tags identify demographic/social group mentions, including neutral and pejorative references.
- Evaluates classifier F1 over repeated trials and human moderation performance in the user study.

## Results and Claims
- Intent-tag enrichment consistently improves machine hate classification and cross-dataset generalization more than group tags alone.
- Human moderators given intent annotations outperform machine classifiers by a reported 12.9% and rate intent tags as the most useful explanation.
- Group tags are still relevant, but the strongest evidence is for intent-focused enrichment.

## Limitations and Follow-ups
- Tags are generated by GPT-4o and then learned by NER models, so tag quality and domain transfer need checking.
- The user study uses curated IHC examples; deployment performance may differ on noisier streams.
- Follow-up use: compare intent tags with target-group spans from [[154-boudraa-2025-implicit-hate-target-span-identification-in-zero-and-few-shot-settings-with-selective-sub-billion-parameter-models]] and with rationale approaches in [[042-hare-explainable-hate-speech-detection-with-step-by-step-reasoning]].
