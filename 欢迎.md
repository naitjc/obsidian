# 欢迎

这是你的研究维基。

## 快速开始

1. **添加源文件** → 放入 `raw/sources/` 目录
2. **告诉 LLM** → "请处理这个源文件"
3. **提问** → LLM 会从维基中合成答案
4. **存档** → 有价值的答案可以存为新的 wiki 页面

## 目录

- [AGENTS.md](AGENTS.md) - 维基架构和约定
- [wiki/index.md](wiki/index.md) - 内容索引
- [log.md](log.md) - 活动时间线
- [raw/sources/](raw/sources/) - 源文件目录

## 原则

- 源文件是**不可变的** — 只读，从不修改
- 维基是**LLM 维护的** — LLM 创建和更新所有页面
- 你负责**策展** — 选择源文件、提出好问题、思考意义
- 知识是**累积的** — 每次添加都在建立，而非重零开始
